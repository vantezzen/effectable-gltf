# Basic Usage

## The EffectableGltf Component

The `EffectableGltf` component is a drop-in replacement for `@react-three/drei`'s `Gltf` component. It accepts all the same props as the original component, plus the ability to add effects as children.

```tsx
import { EffectableGltf } from "@vantezzen/effectable-gltf";

function MyScene() {
  return (
    <EffectableGltf src="/path/to/model.gltf" position={[0, 0, 0]} scale={1} />
  );
}
```

## Adding Effects

Effects are added as children to the `EffectableGltf` component. You can add multiple effects, and they will be applied in the order they are specified.

### Overlay Effect

The `Overlay` effect adds a color overlay to your model with customizable opacity:

```tsx
import { EffectableGltf, Overlay } from "@vantezzen/effectable-gltf";

function MyScene() {
  return (
    <EffectableGltf src="/path/to/model.gltf">
      <Overlay color="red" opacity={0.5} />
    </EffectableGltf>
  );
}
```

### Outline Effect

The `Outline` effect adds a cartoon-style outline to your model:

```tsx
import { EffectableGltf, Outline } from "@vantezzen/effectable-gltf";

function MyScene() {
  return (
    <EffectableGltf src="/path/to/model.gltf">
      <Outline />
    </EffectableGltf>
  );
}
```

### Combining Effects

You can combine multiple effects on the same model:

```tsx
import { EffectableGltf, Overlay, Outline } from "@vantezzen/effectable-gltf";

function MyScene() {
  return (
    <EffectableGltf src="/path/to/model.gltf">
      <Overlay color="red" opacity={0.5} />
      <Outline />
    </EffectableGltf>
  );
}
```

## Effect Props

### Overlay Props

| Prop    | Type   | Default | Description                                    |
| ------- | ------ | ------- | ---------------------------------------------- |
| color   | string | 'red'   | The color of the overlay (any valid CSS color) |
| opacity | number | 0.5     | The opacity of the overlay (0-1)               |

### Outline Props

| Prop      | Type   | Default | Description                  |
| --------- | ------ | ------- | ---------------------------- |
| color     | string | 'black' | The color of the outline     |
| thickness | number | 0.01    | The thickness of the outline |
| opacity   | number | 1       | The opacity of the outline   |

## Best Practices

1. **Performance**: Effects are applied in order, so consider the performance impact of multiple effects
2. **Opacity**: When using the `Overlay` effect, keep the opacity below 1 to maintain visibility of the original model
3. **Outline Thickness**: Adjust the outline thickness based on your model's scale and camera distance
