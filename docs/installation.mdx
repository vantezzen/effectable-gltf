# Installation

## Prerequisites

Before installing effectable-gltf, make sure you have the following dependencies in your project:

- React (16.8 or higher)
- React Three Fiber
- Three.js

## Package Installation

You can install effectable-gltf using your preferred package manager:

```bash
# Using npm
npm install effectable-gltf

# Using yarn
yarn add effectable-gltf

# Using pnpm
pnpm add effectable-gltf
```

## Peer Dependencies

effectable-gltf requires the following peer dependencies:

```json
{
  "peerDependencies": {
    "react": ">=16.8.0",
    "react-dom": ">=16.8.0",
    "@react-three/fiber": ">=8.0.0",
    "@react-three/drei": ">=9.0.0",
    "three": ">=0.150.0"
  }
}
```

## TypeScript Support

The library includes TypeScript definitions out of the box. No additional type packages are required.

## Next Steps

Once you've installed the package, you can:

1. Import the components in your React application
2. Start using the `EffectableGltf` component
3. Apply built-in effects or create custom ones

Check out the [Basic Usage](/docs/basic-usage) guide to learn how to use the library in your project.
